# Ocular-Trauma-Oncall-Checklist
Getting ready for an ocular trauma consult or medical rotation in ophthalmology? I built a complete, evidence-based copy&amp;paste template so you never forget a step.

OCULAR TRAUMA OPH FINDING TEMPLATE
Mechanism of injury: Blunt trauma (fist/ball/fall/MVC) / Sharp trauma / Mixed / Foreign body / Burn / Bite
VA: Document baseline VA; poor vision (inability to count fingers >1-2 feet) is associated with severe injury (OR 14.5). 
Pupils: to mm, prompt/sluggish/fixed, symmetric/asymmetric light reaction. Assess for RAPD, the strongest predictor of severe injury (OR 44.8). 
EOM: full and free / restricted in which gaze / no motion pain / motion pain present
Gross Exam: Measure baseline proptosis (Hertel) for orbital compartment syndrome monitoring. Check for oculocardiac reflex (bradycardia/nausea vomiting/loss of consciousness), which suggests muscle entrapment.
IOP : Perform serial IOP checks (q2h if OCS suspected, daily for hyphema)
Eyelid: ecchymosis/ laceration / ptosis
Conjunctiva: subconjunctival hemorrhage (extent: ___%) / chemosis / foreign body / laceration
Cornea: clear / epithelial defect (size: __ x mm, location: ____) / Seidel test (-/+) / edema (MCE/stromal) / foreign body / laceration Always perform Seidel test if open globe suspected; document fluorescein staining pattern.
AC: deep / shallow / flat / clear / cell + / flare + / hyphema (layered mm or % / microhyphema / total) / hypopyon / fibrin Assess for phacomorphic (swollen lens, shallow AC) vs phacolytic glaucoma (hypermature lens, deep AC, white particles)
Lens: clear / NS+ / cortical opacity / PSC / traumatic cataract (rosette / anterior subcapsular / cortical / nuclear) / Vossius ring / white cortical material in AC / intumescent (phacomorphic) / capsular rupture / phacodonesis / subluxation (direction: ___) / dislocation (anterior chamber / vitreous)
Vitreous: media clear / Weiss ring / vitreous opacities (VO) / vitreous hemorrhage (mild/moderate/severe, obscuring view) / pigment (Shafer sign) / anterior chamber opacities (ACO)  *Weiss ring = complete PVD; Shafer sign (pigment cells) suggests retinal break; ACO suggests zonular damage 
Fundus: C/D % / disc edema / disc pallor / disc avulsion / peripapillary hemorrhage / commotio retinae (location: ) / retinal whitening / choroidal rupture / retinal hemorrhage (location/pattern: ) / retinal break (location: o'clock, size: ) / retinal detachment (extent: _ quadrants, macula on/off) / macular hole / RPE changes / subretinal fluid / attached
Ishihara: passed and prompt
*Color vision defect suggests optic neuropathy

Additional:
Gonioscopy (defer if hyphema present): Angle open / narrow / closed / angle recession (extent: clock hours, location: ) / peripheral anterior synechiae (PAS) / trabecular pigmentation / cyclodialysis cleft
Facial sensation: Intact / Decreased in V1/V2/V3 distribution 
*Infraorbital nerve injury common with orbital floor fractures
